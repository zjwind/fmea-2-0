```
  ▼                                                                        │

  │ 项目管理 → DRBFM触发评估 → DRBFM分析 → 基线输出与落地跟踪               │

  │ 评审与审批 → 入库管理 → DRBFM迭代更新                                    │

  │ 基线库与知识图谱 → 看板与统计                                            │
  └────────────────────────────────────────────────────────────────────────┘
```

## 8. 关键路径与风险

### 8.1 关键路径

```
通用基础 → 项目管理 → 触发评估 → DRBFM分析 → 基线输出 → 评审审批 → 入库管理
```

关键路径上的任何延迟都会直接影响整体交付。

### 8.2 风险项

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| AI模型接口不稳定 | DRBFM分析模块阻塞 | AI调用全部异步+降级方案(手动填写) |
| 飞书画板API限制 | 结构分析模块阻塞 | iframe嵌入方案已降低耦合，画板创建失败不阻塞其他步骤 |
| PMS数据源连接不稳定 | 项目管理/触发评估阻塞 | Redis缓存30分钟+降级提示 |
| SOD模式1/2差异未明确 | SOD评分模块 | 一期按统一规则实现，预留modeType参数 |
| 落地失败重新分析未定义 | 基线跟踪模块 | 预留接口，当前仅记录状态 |

### 8.3 可并行开发项

以下模块在满足依赖条件后可并行开发：

| 并行组 | 模块 | 前置依赖 |
|--------|------|----------|
| 并行组A | DRBFM迭代更新 + 入库管理 | 评审审批完成 |
| 并行组B | 基线库 + 看板统计 | 入库管理完成 |
| 并行组C | 系统配置 + 外部集成 | 无（阶段0内并行） |
